## Starting point for a PWA x Mobile App using Svelte, TailwindCss & Capacitor

### Install:<br>
`npm install`

### For web:<br>
`npm run dev`

### For Android:<br>
`npx cap add android`<br>
`npx cap open android` // open project in Android Studio<br>
`npx cap sync` // sync changes to android<br>
